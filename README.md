PROIECT - Sistem pentru gestionarea unei retele de Sali de Fitness - creat de Ghita Andreea si Gheablau Nicoletta;

DESCRIERE - Aplicatia gestioneaza o retea de sali de fitness, permitand utilizatorului sa cumpere abonamente, sa rezerve participarea la clase si sa isi urmareasca activitatea proprie.
Aplicatia contine doua tipuri de utilizatori : Administrator si Client si accesul la functionalitati se face abia dupa autentificare.

ADMINISTRATOR - administrarea salilor si zonelor : creare, modificare a salilor sau zone; configurarea programului si capacitatii zonelor 
- definirea abonamentelor : creare, stergere, modificare abonamente; setarea preturilor si beneficiilor;
- gestionarea claselor si antrenorilor : creare/modificare clase , asociere antreprenori, vizualizare rezervari;
 - monitorizarea activitatatii : statistici despre abonamente active si gradul de ocupare al claselor.

CLIENT - vizualizarea salilor si ofertelor : lista salilor si abonamentelor disponibile; 
- cumpararea unui abonament : activarea si utilizarea abonamentului pe durata valabilitatii;
- rezervarea participarii la clase : vizualizarea programului si rezervarea locurilor;
- gestionarea activitatii personale : permisiunea de a vizualiza clasele rezervate, cat si a abonamentelor active;
- anularea rezervarilor 

PERSISTENTA DATELOR - datele sunt salvate si incarcate automat.
